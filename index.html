Chicago Grocery Walkability Map

Explore grocery store access across Chicago.

About This Project

This project creates an interactive map showing how far someone can walk (about 12 minutes) to reach a grocery store in Chicago.

Features

  • Interactive map
  • Chicago grocery store locations
  • 12-minute walking distance visualization
  • Built using Python and Folium

How It Works

  1. Python downloads grocery store data.
  2. Walking areas are calculated.
  3. Folium creates an interactive map.
  4. The map is saved as docs/index.html.
Built with Python, Folium, and GitHub Pages
