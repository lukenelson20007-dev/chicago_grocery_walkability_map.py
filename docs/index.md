# Chicago Grocery Walkability Map

This project visualizes the accessibility of grocery stores across Chicago neighborhoods using a 12-minute walk radius.

## About

The map shows which areas of Chicago have good walkable access to grocery stores and which neighborhoods may have limited options for purchasing fresh food.

## How to Use

1. Explore the interactive map
2. Check the walkability score for different neighborhoods
3. View nearby grocery stores within a 12-minute walk

## Data Sources

- Chicago grocery store locations
- Neighborhood boundaries
- OpenRouteService for accessibility calculations
